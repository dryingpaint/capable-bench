# Task Bundles

Curated benchmark task directories and `problems.csv` live here.

Task data are gitignored by default because they may contain private or derived
scientific data. Commit selected anonymized task sets only after review.

BioDiscoveryBench task bundles may include peptide pharmacology tables,
in vivo summaries, omics tables, structures, generated candidates, or
foundation-model outputs. Keep the task-local files limited to evidence the
agent is allowed to see; hidden outcomes and expert rubrics belong in
`data/answers/`.
